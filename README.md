# XinFramework

根据个人工作总结，结合三方开源代码，符合国内设计风格，做一个简单、快速的开发框架，可以拿来就搭建新的项目。由于工作比较忙只能闲暇时间来做，2017年6月第一次提交，后续功能不断完善和持续更新中。我希望能够帮助你同时希望有大神指教。
